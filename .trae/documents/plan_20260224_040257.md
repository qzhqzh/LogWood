## LogWood MVP 实现计划

### Phase 1: 项目初始化 (Week 1)
1. **初始化 Next.js 14 项目**
   - 配置 TypeScript + Tailwind CSS
   - 配置 ESLint + Prettier
   
2. **数据库层**
   - 配置 Prisma + PostgreSQL
   - 创建完整 Schema（7 个核心表 + 枚举）
   - 执行数据库迁移

3. **认证系统**
   - 配置 NextAuth.js
   - 实现邮箱魔法链接登录
   - 实现 GitHub OAuth（可选）
   - 实现匿名用户设备指纹识别

### Phase 2: 核心模块 API (Week 2-3)
4. **Identity 模块**
   - `resolveActor()` 身份解析
   - `ensureAnonymous()` 匿名用户创建

5. **Target 模块**
   - 工具目录 API
   - 工具详情 API
   - 初始化种子数据（Cursor, Windsurf, Copilot 等）

6. **Rate-Limit 模块**
   - 限流检查与计数
   - 配置化限额管理

7. **Review 模块**
   - 发布评测 API
   - 评测列表 API（排序/筛选）
   - 评测详情 API

8. **Like 模块**
   - 点赞 API（幂等）
   - 点赞统计（匿名/登录比例）

9. **Comment 模块**
   - 评论发布 API
   - 评论列表 API

10. **Moderation 模块**
    - 内容风险评估
    - 举报 API
    - 自动折叠逻辑

### Phase 3: 前端页面 (Week 3-4)
11. **布局与导航**
    - 全局 Layout
    - 导航栏 + 页脚

12. **首页**
    - 热门评测
    - 最新评测
    - 分类入口

13. **工具目录页**
    - `/editor` 编辑器列表
    - `/coding` 编程助手列表

14. **工具详情页**
    - `/:type/[slug]`
    - 工具信息 + 评测流

15. **评测详情页**
    - `/review/[id]`
    - 点赞 + 评论 + 举报

16. **提交评测页**
    - `/submit`
    - 表单 + 校验

17. **用户中心**
    - `/profile`
    - 我的评测/点赞

### Phase 4: 测试与验收 (Week 4-5)
18. **单元测试**
    - 核心业务逻辑测试

19. **集成测试**
    - API 端点测试

20. **E2E 测试**
    - 关键用户流程测试

---

**技术栈**: Next.js 14 + TypeScript + Prisma + PostgreSQL + NextAuth.js + Tailwind CSS

**预计时间**: 4-6 周完成 MVP
